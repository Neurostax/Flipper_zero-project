# Schematics & PCB Design

This project uses KiCad for its hardware design. You can find the actual schematic and PCB design files in the `hardware/` directory at the root of the project:

- **Schematic**: `hardware/schematic.kicad_sch`
- **PCB**: `hardware/pcb.kicad_pcb`

Please refer to those files for detailed pin mappings and circuit layout.
